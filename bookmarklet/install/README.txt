PORTMARKS BOOKMARKLET v1.0.0
============================

WHAT IS PORTMARKS?
Portmarks creates portable HTML bookmark files for any web page. These bookmarks
work everywhereâ€”no browser sync or cloud dependency required.

INSTALLATION INSTRUCTIONS

Desktop:
1. Open the `install.html` file in your web browser.
2. Drag the "Portmarks" button to your bookmarks bar, or right-click the button
and select "Bookmark Link" (or a similar option).

Mobile:
Option 1 - Sync from Desktop (Easiest):
- Install on your desktop first, then enable bookmark sync in your browser. The
bookmarklet will automatically appear in your mobile bookmarks.

Option 2 - Manual Installation:
1. Open the `install.html` file in your mobile browser.
2. Long-press the "Portmarks" button and select "Bookmark Link" (or a similar
option).
3. If that doesn't work: Create any bookmark, then edit it and replace the URL
with the bookmarklet code from the `bookmarklet.uri.txt` file.

USAGE
1. Navigate to any web page you want to bookmark.
2. Click the "Portmarks" bookmark in your bookmarks bar.
3. A portable HTML bookmark file will be downloaded automatically.
4. Open the downloaded file anytime to return to that web page.

SYSTEM REQUIREMENTS
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- Works on Windows, Mac, Linux, Android, and iOS
- No additional software required

TROUBLESHOOTING
If the Portmarks bookmarklet does not download the bookmark file, try using it
on a different website to ensure it's not a site-specific issue. Some sites
(especially those with strict security policies) may prevent file downloads
from bookmarklets.
